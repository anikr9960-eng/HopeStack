# HopeStack — AI Mental Health Companion

HopeStack is a premium, mobile-first mental health journal and wellness application designed to support your emotional well-being through smart technology and clinically-validated tools.

## ✨ Features
- **Personalized Auth**: Unified Login/Signup with session persistence.
- **Multilingual Support**: Seamlessly switch between **English**, **Hindi**, and **Bengali**.
- **AI Sentiment Engine**: VADER-powered journal analysis with support for Gen-Z slang (e.g., "slay", "rizz", "no cap").
- **Mood Tracking**: Visualized trends, sparklines, and a Github-style consistency heatmap.
- **Wellness Tools**: PHQ-9 depression screening, animated 4-7-8 breathing exercises, and randomized affirmations.
- **Reliable Resources**: Hand-picked articles and guides from **NIMH**, **Mayo Clinic**, and the **APA**.
- **Privacy-First**: No backend; all data stays securely in your browser's `localStorage`.

## 🛠️ Technology Stack
- **Structure**: Semantic HTML5
- **Style**: Custom Vanilla CSS3 (Glassmorphism design system)
- **Logic**: Vanilla JavaScript ES6+
- **Charts**: Custom Canvas-based visualizations

## 🚀 Getting Started
1. Clone this repository:
   ```bash
   git clone https://github.com/anikr9960-eng/HealthApp.git
   ```
2. Open `index.html` in any modern web browser.
3. No installation or Node.js required! 🌿✨
